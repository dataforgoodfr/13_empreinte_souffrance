# Intégrer la carte interactive sur votre site

La carte affiche les supermarchés en France avec des marqueurs indiquant la présence ou l'absence d'œufs de poules en cage. Les utilisateurs peuvent filtrer par enseigne et par statut.

  - **URL de la carte dans le site web :** `https://lheuredescomptes.org/numbers#carte`
  - **URL de la carte version embed :**  `https://lheuredescomptes.org/embed/store-map`

## Intégration

Copiez-collez ce code HTML dans votre page :

```html
<iframe
  src="https://lheuredescomptes.org/embed/store-map"
  width="700"
  height="700"
  title="Carte des supermarchés - L'heure des comptes"
></iframe>
```
